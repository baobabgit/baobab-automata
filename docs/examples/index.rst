Structure des Exemples
-----------------------

Chaque exemple comprend :

* **Description** : Explication du problème résolu
* **Code complet** : Implémentation fonctionnelle
* **Exécution** : Instructions pour tester l'exemple
* **Résultats attendus** : Sortie et visualisations
* **Variations** : Modifications possibles

Comment utiliser ces exemples
-----------------------------

1. **Copiez le code** dans un fichier Python
2. **Installez les dépendances** si nécessaire
3. **Exécutez le script** et observez les résultats
4. **Modifiez les paramètres** pour explorer les variations
5. **Consultez la documentation API** pour comprendre les détails

Conseils pour les débutants
----------------------------

* Commencez par les exemples d'automates finis
* Testez chaque exemple avant de passer au suivant
* Modifiez légèrement les paramètres pour comprendre leur impact
* Utilisez la visualisation pour mieux comprendre le comportement

Conseils pour les utilisateurs avancés
--------------------------------------

* Explorez les algorithmes d'optimisation
* Testez les performances sur de gros automates
* Intégrez Baobab Automata dans vos projets existants
* Contribuez vos propres exemples à la communauté
